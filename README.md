# Support Ticketing System

## Novi Update

### 1. Zaštita pristupa admin stranici
- Dodana autentifikacija za pristup `admin.php`. Samo administratori mogu pristupiti tim stranicama.

### 2. Gumb "Otkazi zahtjev" sakriven za otkazane zahtjeve
- Ako je zahtjev već otkazan, gumb "Otkazi zahtjev" se više ne prikazuje.

### 3. Filtriranje otkazanih zahtjeva na admin stranici
- Otkazani zahtjevi više nisu prikazani na admin stranici.

### 4. Dodan razlog otkazivanja za administratore
- Ako admin otkazuje zahtjev, mora unijeti razlog otkazivanja.

### 5. Ispravljena greška s kosim crtama u tekstu zahtjeva
- Kose crte više neće biti duplirane pri spremanju zahtjeva.
